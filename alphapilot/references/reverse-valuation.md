# 逆向定价与市值倒推

## 目标

逆向定价用于回答：当前市值到底要求公司未来做到什么程度。它不是替代五步法，而是防止把“好公司”和“好价格”混为一谈。

## 必做场景

满足任一条件时必须做逆向定价：

| 条件 | 阈值 |
|---|---:|
| PE-TTM | > 80 |
| PS-TTM | > 20 |
| 市值/年化收入 | > 50 |
| 单季利润同比增长 | > 100% |
| 一年股价涨幅 | > 200% |
| 净利润大增但经营现金流明显跟不上 | 净现比 < 0.5 |

## 核心计算

| 项目 | 公式 |
|---|---|
| 总市值 | 当前股价 × 总股本 |
| TTM 净利润 | 最近四个季度归母净利润合计 |
| PE-TTM | 总市值 / TTM 净利润 |
| PS-TTM | 总市值 / TTM 营收 |
| 年化收入 | 最近单季营收 × 4 |
| 年化利润 | 最近单季净利润 × 4 |
| 目标 PE 隐含全年利润 | 总市值 / 目标 PE |
| 下一季度验证利润 | 目标半年利润 - 已披露季度利润 |
| 下一季度验证营收 | 下一季度验证利润 / 假设净利率 |

## 结论写法

必须明确区分：

- 公司是否有增长逻辑。
- 当前价格是否已经提前反映增长。
- 下一季度需要交出什么财务结果，当前估值才站得住。

避免写法：公司处于高景气赛道，因此值得买入。

推荐写法：公司处于高景气赛道，但当前市值要求下一季度营收达到 X 亿元、净利润达到 Y 亿元；若低于该验证线，估值将面临压缩。
