# 行业关键因子模板

## 使用规则

报告必须列出影响公司定价逻辑的 5-10 个关键因子。若行业不在下表中，需自行归纳并标注信息源。

## 内置行业模板

| 行业 | 关键因子 | 主要信息源 |
|---|---|---|
| 光模块/光芯片 | 云厂商 capex、800G/1.6T、CW/EML 供需、同行毛利率、应收账款、客户认证、技术路线 | 公司公告、Coherent/Lumentum/Fabrinet 财报、NVIDIA/云厂商财报、C114、讯石 |
| 储能 | 碳酸锂价格、电芯价格、海外库存、订单质量、毛利率、产能利用率、政策补贴 | 公司公告、SMM、行业协会、海关数据、海外电力市场数据 |
| 消费 | 同店增长、渠道库存、毛利率、费用率、现金回款、经销商数量、客单价 | 年报、渠道调研、统计局、公司经营数据 |
| 券商 | 成交额、两融余额、IPO/再融资、权益市场风险偏好、自营收益、佣金率 | 交易所、证券业协会、公司财报、市场成交数据 |
| 半导体设备 | 国产替代率、订单 backlog、验收周期、客户扩产、研发费用率、毛利率 | 公司公告、客户扩产公告、SEMI、产业链调研 |

| 因子 1 | | | | |

## 解释要求

每个因子必须说明：

1. 为什么影响估值。
2. 观察频率是季度、月度还是事件驱动。
3. 哪个变化会强化逻辑。
4. 哪个变化会推翻逻辑。
