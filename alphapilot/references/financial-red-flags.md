# 财务排雷规则

## 核心原则

利润不是现金。高增长公司尤其要检查收入、利润、现金流、应收账款和存货是否匹配。

## 排雷指标

| 指标 | 公式或观察口径 | 绿灯 | 黄灯 | 红灯 |
|---|---|---:|---:|---:|
| 净现比 | 经营现金流净额 / 净利润 | > 0.7 | 0.3-0.7 | < 0.3 |
| 销售回款率 | 销售商品收到现金 / 营业收入 | > 1.0 | 0.8-1.0 | < 0.8 |
| 应收账款/单季营收 | 期末应收账款 / 单季营收 | < 1.0 | 1.0-1.3 | > 1.3 |
| 存货/单季营收 | 期末存货 / 单季营收 | 行业合理 | 快于收入 | 明显堆积 |
| 毛利率异常 | 当前毛利率 vs 历史/同行 | 稳定 | 明显偏高 | 高位快速回落 |
| 净利率异常 | 当前净利率 vs 历史/同行 | 稳定 | 明显偏高 | 依赖一次性收益 |
| 大存大贷 | 货币资金与有息负债同时畸高 | 无 | 需解释 | 存款利息异常低 |

## 一票否决或强预警

出现以下情况必须显著降低评级：

1. 净现比连续两年低于 0.5。
2. 经营现金流为负且应收账款继续大幅增长。
3. 收入高增长但销售回款率持续低于 0.8。
4. 存货增长显著快于收入增长且没有明确订单解释。
5. 毛利率、净利率显著高于同行但公司没有披露可验证原因。
6. 货币资金高、有息负债高、利息收入低同时出现。

## 报告写法

财务排雷必须给出“通过/预警/否决”结论，不能只罗列数据。若触发红灯，必须说明它如何影响估值或持仓策略。
